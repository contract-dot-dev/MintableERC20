# MintableERC20

This repo contains a Hardhat V2 project with a `MintableERC20` token contract.
It’s designed as a demo repo for use with [contract.dev](https://contract.dev).

Connect it to a Stagenet to automatically import the contract and generate a dedicated Contract Workspace with built-in analytics and tooling.

## Get Started

Follow this [quickstart](https://docs.contract.dev/get-started/hardhat/v2) to import this repo’s ERC20 contract onto a Stagenet.
